# multivariable-calculus

Flashcards for Multivariable Calculus — for spaced repetition review via [flashcards](https://github.com/thomasrribeiro/flashcards).

## Coverage

| File | Topic |
|------|-------|
| 01_vectors_3d | Vectors in $\mathbb{R}^3$, dot product, cross product, projections |
| 02_lines_planes_surfaces | Lines, planes, quadric surfaces, cylindrical/spherical coordinates |
| 03_vector_valued_functions | Vector-valued functions, limits, derivatives, integrals of $\vec{r}(t)$ |
| 04_arc_length_curvature | Arc length, reparametrization, $\hat{T}$, $\hat{N}$, $\hat{B}$, curvature, torsion |
| 05_multivariable_limits | Functions of several variables, limits, continuity, path test |
| 06_partial_derivatives | Partial derivatives, higher-order, Clairaut's theorem |
| 07_chain_rule_gradient | Multivariable chain rule, gradient, directional derivative |
| 08_tangent_planes_linearization | Tangent planes, linearization, differentials, differentiability |
| 09_extrema_lagrange | Critical points, second derivative test, Lagrange multipliers |
| 10_double_integrals | Double integrals, iterated integrals, Fubini, polar coordinates |
| 11_triple_integrals | Triple integrals, cylindrical, spherical coordinates |
| 12_change_of_variables | Jacobian, change of variables in double and triple integrals |
| 13_vector_fields_line_integrals | Vector fields, scalar/vector line integrals, work, flux in 2D |
| 14_greens_theorem | Conservative fields, FTLI, path independence, Green's Theorem |
| 15_surface_stokes_divergence | Surface integrals, flux, curl, divergence, Stokes' and Divergence theorems |
